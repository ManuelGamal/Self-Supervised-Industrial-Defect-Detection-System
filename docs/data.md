# Data Pipeline Documentation

## Dataset: MVTec Anomaly Detection (MVTec AD)

**Source:** [MVTec AD Website](https://www.mvtec.com/company/research/datasets/mvtec-ad)
**Kaggle:** Uploaded as a private dataset (6-category subset)

### Selected Categories

| Category | Train (normal) | Test (normal) | Test (defect) | Total |
|----------|---------------|---------------|---------------|-------|
| bottle   | 209           | 20            | 63            | 292   |
| capsule  | 219           | 23            | 109           | 351   |
| carpet   | 280           | 28            | 89            | 397   |
| hazelnut | 391           | 40            | 70            | 501   |
| leather  | 245           | 32            | 92            | 369   |
| pill     | 267           | 26            | 141           | 434   |

### Directory Structure

```
mvtec-ad/
  <category>/
    train/
      good/          # Normal training images (*.png)
    test/
      good/          # Normal test images
      <defect_type>/ # Anomalous test images (one folder per defect)
    ground_truth/    # Pixel-level segmentation masks
```

## Split Strategy

Splits are generated by `src/data/splits.py` and stored in `data/splits/`.

### Configuration

- **3 ratio configurations** applied to each of the 6 categories:
  - `70/15/15` — 70% train, 15% val, 15% test
  - `80/10/10` — 80% train, 10% val, 10% test
  - `60/20/20` — 60% train, 20% val, 20% test
- **Fixed seed:** 42 (reproducible)
- **Stratified splitting:** both normal and anomalous samples are proportionally distributed across splits
- **Total CSVs generated:** 6 categories x 3 ratios = **18 CSV files**

### CSV Schema

| Column       | Type | Description                              |
|-------------|------|------------------------------------------|
| `image_path` | str  | Relative path from MVTec root            |
| `label`      | int  | 0 = normal, 1 = anomalous               |
| `split`      | str  | One of: `train`, `val`, `test`           |
| `category`   | str  | MVTec category name                      |

### Naming Convention

```
data/splits/<category>_ratio<train>_<val>_<test>_seed<seed>.csv
```

Example: `bottle_ratio70_15_15_seed42.csv`

### Regenerate Splits

```bash
python -m src.data.splits \
  --root path/to/mvtec-ad \
  --out data/splits \
  --seed 42
```

## Augmentation Pipelines

Defined in `src/data/augmentations.py`.

### Train Transform (supervised fine-tuning)

| Step | Details |
|------|---------|
| Resize | 224 x 224 |
| Random Horizontal Flip | p=0.5 |
| Color Jitter | brightness=0.3, contrast=0.3 |
| Gaussian Blur | kernel=3, sigma=(0.1, 0.5) |
| ToTensor | [0, 1] float32 |
| ImageNet Normalize | mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225] |

### SSL Transform (contrastive pretraining)

| Step | Details |
|------|---------|
| Random Resized Crop | 224 x 224, scale=(0.2, 1.0) |
| Random Horizontal Flip | p=0.5 |
| Color Jitter (random apply, p=0.8) | brightness/contrast/saturation/hue |
| Random Grayscale | p=0.2 |
| Gaussian Blur | kernel=23 (0.1 * 224, odd) |
| ToTensor + ImageNet Normalize | Same as above |

### Eval Transform (inference)

| Step | Details |
|------|---------|
| Resize | 224 x 224 |
| ToTensor | [0, 1] float32 |
| ImageNet Normalize | Same as above |

## Data Module

`src/data/datamodule.py` provides a **PyTorch Lightning DataModule** that:

1. Reads a single split CSV
2. Filters rows by category and split (train/val/test)
3. Applies the appropriate transforms
4. Returns configured DataLoaders with a custom collate function

### Usage

```python
from src.data import MVTecDataModule

dm = MVTecDataModule(
    root="path/to/mvtec-ad",
    category="bottle",
    split_csv="data/splits/bottle_ratio70_15_15_seed42.csv",
    batch_size=32,
    num_workers=4,
)
dm.setup()

for batch in dm.train_dataloader():
    images = batch.image      # (B, 3, 224, 224)
    labels = batch.label      # list[int]
    categories = batch.category  # list[str]
```

## How to Reproduce

1. **Download the dataset** from Kaggle (private, 6-category subset) or from the [MVTec AD website](https://www.mvtec.com/company/research/datasets/mvtec-ad)
2. **Generate splits:**
   ```bash
   python -m src.data.splits --root path/to/mvtec-ad --out data/splits
   ```
3. **Run tests:**
   ```bash
   python -m pytest tests/test_data.py -v --cov=src/data
   ```
